## Descrição do Projeto
Este projeto consiste no desenvolvimento de um sistema de autoatendimento para restaurantes e lancherias, permitindo que clientes realizem pedidos de forma automatizada através de um quiosque digital.

O sistema foi desenvolvido em Java com interface gráfica utilizando JavaFX, seguindo a arquitetura MVC (Model, View e Controller).

## Objetivo
O objetivo do projeto é simular o funcionamento de um quiosque de autoatendimento, permitindo:

- Iniciar atendimento
- Selecionar produtos
- Adicionar itens ao pedido
- Visualizar valor total
- Finalizar pedido
